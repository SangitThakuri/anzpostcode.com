# ANZPostcode.com

A premium, fully static Australia and New Zealand postcode directory built with Next.js.

## Tech Stack

- **Next.js 16** App Router with TypeScript
- **Tailwind CSS v4** with custom design system
- **Fuse.js** for client-side search
- **Leaflet + OpenStreetMap** for maps
- **Lucide Icons** for UI icons
- **GitHub Pages** for hosting

## Features

- 18,000+ Australian postcodes and suburbs
- 160+ New Zealand postcodes and localities
- Client-side search with autocomplete
- Interactive OpenStreetMap maps on every location page
- SEO-optimised static pages with JSON-LD schema
- FAQ sections on every postcode/suburb page
- Social sharing (Web Share API, Facebook, WhatsApp, X, LinkedIn)
- Responsive, mobile-first design

## Development

\`\`\`bash
npm install
npm run dev
\`\`\`

## Build

\`\`\`bash
npm run build
\`\`\`

Outputs to `./out/` as static HTML files ready for GitHub Pages.

## Data

- Australian postcodes: community-maintained dataset (MIT license)
- New Zealand postcodes: open geographic data
